# AEDII

Repositório dedicado à disciplina de **Algoritmos e Estruturas de Dados II**.

## 📋 Descrição

Este projeto contém implementações e estudos práticos de algoritmos avançados e estruturas de dados complexas em Java.

## 💻 Tecnologias

- **Linguagem:** Java
- **Tema:** Estruturas de Dados Avançadas e Algoritmos

## 📚 Conteúdo

Este repositório inclui:
- Implementações de estruturas de dados avançadas
- Algoritmos complexos e otimizados
- Análise de complexidade
- Exemplos práticos e exercícios
